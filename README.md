# Bioinformatics & Data Analysis Portfolio

This repository contains sample codes of selected bioinformatics and data analysis tasks
demonstrating my experience in NGS data processing, genome analysis,
and reproducible computational workflows.


## Technical Skills Demonstrated

- **Programming / Scripting**: Bash, Python, Jupyter Notebook, R, Perl, Matlab, etc.
- **Bioinformatics Tools**: fastp, SKESA, Parsnp, samtools, etc.
- **Data Analysis**: pandas, numpy, DESeq2, visualization
- **Environments and Version Control**: Conda/Mamba, GitHub


## How to Use This Repository

Each project directory contains:
- A README describing the task
- Scripts for each analysis step
- Example directory structures for input and output data
- Raw sequencing data are not included due to size constraints.
